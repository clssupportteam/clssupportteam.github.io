"Good morning/afternoon/evening, can I speak to {client name}"

"My name is ……. and I’m calling from the Co-op Funeralcare Support Team."

"As you’re arranging a funeral with us {for deceased name} I want to let you know about some support that is included as part of the funeral arrangement to help you deal with some practical matters.

  • [ If Yes ]- Go to 2 - CLS Support

  • [ If Yes ]- "OK, can I confirm/take your email address please?" - Send hub link - Go to 2.2 - Probate check
  • [ If No ]- "OK, thank you." Go to 2.2 - Probate check

2) "The second thing Co-op Legal Services can help with is checking if you need probate."

  • [ If No ]- "Okay, no problem." Go to 5 - Close Call
Probate explanation if needed for client

Probate, is the legal process of dealing with someone’s money, property, and possessions after they’ve died.

3 - Probate Need Questions

3.1 - Determine if Probate is required?
[Select any that apply]

  1. "Did {deceased name} own a property in their sole name?"
  2. "Did {deceased name} have any bank or account with more than £30,000 just in their name?"

  • [ If Yes to any of the above questions? ] - The estate likely needs probate - Go to 3.2 - Probate Needed
  • [ If No to 'All' of the above questions? ] - Probate not needed - Go to 5 - Close Call
3.2 - Probate Needed
[ If yes to either of the probate questions ]

"Its likely you’ll need Probate. If you've not already instructed someone, I can ask a specialist from Co-op Legal Services to call you?"

  • [ If Yes ] - Client agrees to consultation - Go to 3.3 - Meeting
  • [ If No ] - Client declines a consultation - Go to 5 - Close Call
3.3 - Meeting
[ If Yes to the consultation ]

"OK, I just need to ask a couple more questions"

  • "Did {deceased name} leave a Will?" -

    • [ If Yes ] - "Are you the Executor named in the Will?"

      • [ If Yes ]: You are speaking with the correct PR - "Okay, just a few more questions..." - Go to 3.4 - £750k question

      • [ If No ] - "OK, for the meeting to go ahead we will need to speak to the executor named in the will. I can provide you the email of support I mentioned earlier, and if you pass this on to the executor, they will be able to request a call back from CLS when it is convenient for them." - Go to 5 - Close Call


    • [ If No ] - "Are you the [deceased name] next of kin?" Apply intestacy rules to establish correct Personal Representative.

      • [ If Yes ]: You are speaking with the correct PR - "Okay, just a few more questions..." - Go to 3.4 - £750k question

      • [ If No ] - "OK, for the meeting to go ahead we will need to speak to the executor named in the will. I can provide you the email of support I mentioned earlier, and if you pass this on to the executor, they will be able to request a call back from CLS when it is convenient for them" - Go to 5 - Close Call
3.4 - £750,000 Question

"To make sure we help you speak to the right person, do know if the sole estate for your [Relation] could be over £750k?"

  • [ If client asks why you need this information] - "To ensure the Consultant can provide a bespoke Fixed Fee Quote that is accurate and in line with the complexities involved with High Value Estates"
  • [If Yes] - The meeting can only be booked with a consultant according to the high value estate booking exceptions list.

"Thank you, I’ll check their diary and let you know about their availability. They’ll go through everything on the call with you, and at the end they will provide a no obligation fixed fee quote for Probate."
